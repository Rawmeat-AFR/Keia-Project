# 1.**鶏鴉プロジェクト　README**

鶏鴉プロジェクトは、魔導と科学が融合した技術体系を持つ完全オリジナルの異世界を舞台とした世界構築プロジェクトである。本資料はプロジェクト全体の基本的な世界観・設定の概要をまとめたものであり、各設定資料を参照する際の基盤となる。

---
# 2.**世界の基本情報**

## 2-1 世界の概要

本プロジェクトの舞台は、現実世界とは独立した完全オリジナルの異世界である。技術水準は現実の2026年を上回る水準にあり、魔導技術と科学技術の両方が高度に発展・融合した社会が形成されている。

## 2-2 技術体系の概要

この世界における技術体系は、大きく以下の二つの柱から成る。

- 科学技術：
  現実世界の科学技術を超える水準で発展した技術体系。
- 魔導技術：
  フリオンと呼ばれるエネルギー粒子を扱う技術体系。科学技術と融合し、現代では両者を組み合わせた技術が広く実用化されている。

---
# 3.**魔導技術の基本**

## 3-1 フリオン（Flion）

魔導技術の根幹をなすエネルギー粒子。余剰次元から継続的に流入しており、電磁気力・強い力・弱い力の三つの基本力に統一的に干渉できる大統一理論的粒子である。科学でいう電気に相当するエネルギー源として、工夫次第でほぼあらゆる用途に応用可能である。詳細は「フリオン基本資料」を参照のこと。

## 3-2 魔導技術の定義

フリオンを扱う技術全体を魔導技術と総称する。現代においては科学技術と融合しており、両者を明確に区別することは実用上ほとんどない。

## 3-3 魔導の発動

魔導の発動には媒介が必要であり、人間が行う場合は金属換装した指・腕を媒介として使用する。空中に複合フリオンで円を形成し、その内側に配列を投影することで現象を発現させる。機械・装置による発動も同様の原理に基づく。

# 4.**インプラント技術**

## 4-1 概要

インプラントとは、体内に埋め込むことでフリオンの操作・制御を可能にする技術デバイスの総称である。インプラントの普及により、魔導技術の使用を個人でできるようになった。

## 4-2 主要なインプラントの種類

インプラントはフリオン輸送系・演算系を主軸として発展し、それに付随する形で多様な種類が派生・発展している。

- 輸送系：
  呼吸または液化フリオンの吸引で取り込んだフリオンを、体内に形成した微弱な電磁場によって血流に乗せ、目的部位へ誘導・輸送する。
- 演算系：
  脳の情報処理を補助し、高速演算を行う。
